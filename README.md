### **About the application**

üêç AI-powered query on databases to automate the process of query, visualization, etc. Organizations can analyze their data and make decisions much faster. Leverages the use of AI and LLM.
The application is user-friendly. The user only needs to enter the API keys, connection strings, etc., from the databases' settings section. They should be good to explore their databases!
